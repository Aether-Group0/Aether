The Spark Language

Spark is a lightweight, high-speed automation language. It is designed to bridge the gap between simple batch scripts and complex programming with native system hooks.

Version 2.0 Updates

  • Multiline Logic: Full support for if/else and while blocks.
  • Code Reusability: Create reusable logic with the new define keyword.
  • Advanced IO: Dynamic string concatenation for the say command.

Getting Spark

Spark is open-source and maintained on GitHub. You can download the latest binaries or build from source.

Variables

Data is stored using the set keyword. Spark is dynamically typed—numbers, strings, and expressions are handled automatically.

set user = "Admin";
set health = 100;
set status = health / 2;
ActionSyntax
Assignmentset [name] = [value];
Mathset x = (10 + 5) * 2;
Variable Usagesay x;

Input & Output

Communicate with users through text-based console output and interactive dialog boxes.

The Say Command

Prints data to the console. Version 2.0 now supports mixing strings and variables in one line:

say "Hello," username "!";

The Ask Command

Interrupts execution to prompt the user for input via a dialog window.

ask age "How old are you?";

Conditions

Direct the flow of your script using if and else. Every block must be terminated with an end keyword.

if score > 50
  say "You passed!";
else
  say "Try again.";
end;

Loops

Use the while loop to repeat a block of code as long as a condition remains true.

set i = 1;
while i < 6 do
  say "Iteration:" i;
  set i = i + 1;
end;

System Control

Spark can interact directly with your Operating System to automate tasks, launch apps, or manage files.

CommandEffect
webOpens a URL in the default web browser.
runExecutes a system command or opens an app.
createCreates a new file at the specified path.
deleteRemoves a file from the system.
web "https://google.com";
create "notes.txt";
run "notepad.exe";

Functions

Encapsulate logic into reusable blocks. Use define to create a function and call to run it.

define startup;
  say "System Initialized...";
  run "spotify.exe";
end_func;

call startup;
